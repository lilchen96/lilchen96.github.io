Windows右键添加cmd命令行
发表于 2019-03-26| 更新于 2020-03-11|技术

  最近常敲命令行，每次都要cd到相关路径，有点麻烦。上网查询了一下，发现windows系统可以通过注册表的方式将cmd命令行添加到右键菜单。这样就可以在指定路径下打开cmd命令行，很方便。
