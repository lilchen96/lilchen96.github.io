什么是 npm

  npm 全称(Node Package Manager)，是一个用 JavaScript 写的包管理工具。安装了 npm 后，需要下载包时，只需要输入 npm 命令，就会去 npm 官方的仓库下载相关包。

npm 和 Node.js

  npm 的发展是跟 Node.js 的发展相辅相成的。Node.js 是由一个在德国工作的美国程序员 Ryan Dahl 写的。他写了 Node.js，但是 Node.js 缺少一个包管理器，于是他和 npm 的作者一拍即合、抱团取暖，最终 Node.js 内置了 npm。
  后来的事情大家都知道，Node.js 火了。随着 Node.js 的火爆，大家开始用 npm 来共享 JS 代码了，到现在 npm 已经成为了前端的标配。

为什么 npm 下载包慢

  经过了上面的介绍，这个问题的答案就显而易见了。npm 的仓库也就是服务器在国外，所以我们在国内使用 npm 下载依赖包会慢。

如何解决下载慢的问题

  淘宝团队做了一个完整 npmjs.org 镜像叫 cnpm，可以用此代替官方版本(只读)，同步频率目前为 10 分钟一次以保证尽量与官方服务同步。
